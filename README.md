# Facebook 貼文文字轉換器

這是一個簡單的網頁工具，專門用來解決 Facebook 和 Instagram 貼文會自動合併空格和移除空行的問題。

## 功能特色

- **保留排版**：自動在空格和換行符號中插入「零寬度空格 (Zero Width Space)」，防止社群平台自動壓縮排版。
    - ✅ 支援多個連續空格（可用於縮排）。
    - ✅ 支援段落間的空行。
- **Emoji 快捷鍵**：內建常用的特殊符號和 Emoji，方便快速插入。
- **一鍵複製**：轉換後可直接點擊按鈕複製到剪貼板。
- **隱私安全**：純前端運作，所有文字處理皆在瀏覽器端完成，不會上傳至任何伺服器。

## 使用方法

1. 打開 `index.html`。
2. 在左側輸入框輸入您的貼文內容。
3. 右側會即時顯示轉換後的結果。
4. 點擊「📋 複製文字」按鈕。
5. 將文字貼上到 Facebook 或 Instagram 發文。

## 技術細節

- **HTML5 / CSS3 / Vanilla JavaScript**
- 無需安裝任何依賴，直接開啟瀏覽器即可使用。
